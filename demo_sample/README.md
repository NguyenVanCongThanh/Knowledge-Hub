# Demo Calculator Project

Đây là một dự án mẫu cực kỳ đơn giản để minh họa khả năng nạp (ingest), phân đoạn (chunking), liên kết tri thức (link building) và truy xuất thông minh của hệ thống Knowledge Hub.

## Cấu trúc thư mục
* `src/calculator.py`: Chứa mã nguồn triển khai Class `Calculator` cùng các phương thức `add`, `subtract` và `log_operation`.
* `docs/specification.md`: Chứa tài liệu đặc tả yêu cầu chức năng (SRS) mô tả nghiệp vụ của hệ thống.
